DSM/ISM 分析报告
=======================

系统元素：
需求, 设计, 采购, 制造, 测试

原始 DSM 邻接矩阵：
           需求     设计     采购     制造     测试
    需求      0      1      0      0      0
    设计      0      0      1      1      0
    采购      0      0      0      1      0
    制造      0      0      0      0      1
    测试      0      0      0      0      0

可达矩阵：
           需求     设计     采购     制造     测试
    需求      1      1      1      1      1
    设计      0      1      1      1      1
    采购      0      0      1      1      1
    制造      0      0      0      1      1
    测试      0      0      0      0      1

ISM 层级划分（先显示顶层/结果层）：
第 1 层：测试
第 2 层：制造
第 3 层：采购
第 4 层：设计
第 5 层：需求

驱动力与依赖力：
- 需求：驱动力 = 5，依赖力 = 1
- 设计：驱动力 = 4，依赖力 = 2
- 采购：驱动力 = 3，依赖力 = 3
- 制造：驱动力 = 2，依赖力 = 4
- 测试：驱动力 = 1，依赖力 = 5

结果解释：
- 顶层/结果因素：测试
- 底层/驱动因素：需求
- 驱动力越高，通常表示该因素对整个系统的影响越强。
- 依赖力越高，通常表示该因素越容易受到其他因素影响。
